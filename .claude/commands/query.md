# /query — wikiに問い合わせる

引数の質問に対し、`wiki/` の内容を参照して回答してください。

## 手順

1. **wiki検索**
   - Grep で `wiki/**/*.md` を質問のキーワードで検索。
   - 関連すると思われるページを Read で読む（複数可）。
   - 引用元のページ名は回答内に明示する（`（出典: [[ページ名]]）` の形式）。

2. **回答**
   - wikiにある情報のみで回答する。憶測や一般知識での補完を行う場合は **明示的に区別**する（「※ wiki外の一般知識: 〜」など）。
   - 該当情報がwikiにない場合は「wikiに該当情報なし」と答え、関連しそうなページを提示する。

3. **書き戻し（任意）**
   - 回答内容が「wikiに残す価値がある新しい知見」を含む場合、最後に
     「この回答を wiki に書き戻しますか？候補: [[ページ名]] / 新規ページ `xxx.md`」
     とユーザーに確認する。
   - ユーザーが了承したら、CLAUDE.md のルールに従って編集する。**ユーザーの了承なく書き戻さない**。
