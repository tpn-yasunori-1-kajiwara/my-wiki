# /lint — wikiの品質チェック

`wiki/` 全体をスキャンし、以下の問題を検出して **レポートのみ** 返してください。**自動修正はしない**（ユーザーの判断を待つ）。

## チェック項目

1. **sources: が空 / 欠落**
   - フロントマターに `sources:` がない、または空配列のページ。

2. **dead source**
   - `sources:` に記載されたPNGが `raw/` に存在しないページ。
   - Glob で `raw/**/*.png` を取得し、参照と突き合わせる。

3. **孤立ページ**
   - どのページからも `[[このページ名]]` でリンクされていないページ。
   - Grep で `\[\[<ファイル名(拡張子なし)>\]\]` を全 wiki/*.md に対し検索。

4. **title / aliases 重複**
   - 異なるファイル間で同じ title または同じ aliases を持つもの。

5. **同義疑い**
   - ファイル名やtitleが非常に近い複数ページ（例: 「データパイプライン」「データ・パイプライン」）。

6. **要レビューフラグ**
   - 本文に `> **要レビュー:**` を含むページ。

7. **frontmatter 不正**
   - 必須項目 (`title`, `sources`, `created`, `updated`) のいずれかが欠けているページ。

## 出力フォーマット

```
## /lint レポート（YYYY-MM-DD）

### 1. sources欠落 (N件)
- wiki/foo.md
- wiki/bar.md

### 2. dead source (N件)
- wiki/baz.md → 参照: 2026-01-01_missing.png（raw/ に存在しない）

...
```

各項目について件数を先に示し、続いてファイル一覧を箇条書きにしてください。
全項目で問題ゼロなら「✓ 問題なし」と表示してください（この場合のみ✓記号を許可します）。

修正提案を求められた場合のみ、対処案を提示してください（コマンド実行前に必ずユーザー承認を取ること）。
