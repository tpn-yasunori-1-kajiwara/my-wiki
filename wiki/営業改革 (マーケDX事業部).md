# 営業改革 (マーケDX事業部)

## 概要

2026 年度のマーケ DX 事業部における重点テーマ。03/31 の幹部会で発表された「**勝てる提案の型化 AI**」構想と、著者が並行で進めている「**営業に Claude Code を持たせる**」対案からなる、二つのアプローチが並走している状態。

## 本文

### 公式構想（事業部発表）— 03/31

- 過去の **営業履歴・取引履歴・提案書・ノウハウ** を入力とする。
- **RAG**（Retrieval-Augmented Generation）等の技術で「**勝てる提案の型**」を学習。
- 営業現場へ「提案を出力する AI システム」として届ける。

### 著者の対案 — 営業武器化支援

03/31 時点で著者が本部長に説明し「それもいいんじゃない？」と肯定的反応を得た案。チームメンバーへの共有が必要、として整理中。

- 営業の PC に **Codex** や **Claude Code** を入れる。
- クライアント情報をもとに **提案や簡単なデモを開発する便利屋** として動く。
- 小さくてもいいので **お金をもらう or 信用を得る** 動きを積む。
- 大型 RAG システムより、現場での **小さな勝ち筋の積み上げ** を重視する立場。

> **要レビュー:** 公式 RAG 提案 AI 路線と、Claude Code 持たせる路線は **排他ではない可能性が高い**（前者は提案骨子、後者は個別案件の手作業効率化）。両立形態の整理は未確定。

### CSM 方針との接続 — 04/14

- 04/14 に **CSM（カスタマーサクセスマネジメント本部）** の方針発表。
- 著者は対案を CSM 文脈にも接続:
  - **クラウド上で Claude を使えるアプリ**（部内で開発中）をフックにする。
  - 利用が拡がったら **最終的に個別 PC に Claude Code を入れる** フローへ移行。
  - 「営業武器化支援」というラベルで提案中。

### 関連する個別案件

- **東芝の件**（04/23） — 削減効果連動の従量課金で運用開始。営業改革の文脈で、AI による業務削減を金銭価値に変換した実例。→ [[東芝従量課金案件]]
- **ブラマケの提案情報収集効率化**（04/22） — Codex + Skills で実装することに決定。担当者の AI リテラシー向上も狙い。
- **全社 AI 推進室のグローバル法改正情報発信高度化**（04/24） — Web からの信頼性高い情報収集が難点として残る。
