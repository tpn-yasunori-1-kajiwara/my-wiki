# サンドボックス

## 概要
[[Claude-Codeのセキュリティ]] における物理的な隔離レイヤー。権限設定だけでは抜け道（例: Bash 経由の `cat` で Read 禁止を回避）が生まれうるため、OS機能やコンテナで「そもそも実行できない/見えない」状態を作る。

## 本文

### OS標準機能による隔離
- **Seatbelt (macOS)**: macOS標準のサンドボックス機構を Claude Code 起動時に被せる。
- **bubblewrap (Linux)**: 軽量なユーザー空間サンドボックス。`bwrap` コマンドで namespace を分離する。

### ファイルシステム制限
- `denyRead` / `allowRead` / `allowWrite` を組み合わせ、機密ファイル群を読めない・触れない状態にする。
- Read 権限だけでなく Bash 側からも到達できないように両側で塞ぐのが原則。

### ネットワーク制限
- 通信を許可するドメインのリスト化が推奨。
- `WebFetch(domain:)` の指定で「GitHub許可」のような曖昧設定をすり抜けられないようにする。

### コンテナによる完全隔離
- **devcontainer**: VS Code Remote Containers 等を使い、開発環境ごと隔離する。
- レベル5（組織展開）での推奨手段。`bypassPermissionsMode` を有効化するのも、こうした仮想環境内に限られる。

### 権限との関係
公式ドキュメントは「権限とサンドボックスは補完的なセキュリティレイヤー」と位置づけている。サンドボックスだけ・権限設定だけでは穴が残るため、両者を併用する。

## 設定例（原典より）

### Linux 側の前提インストール

`bubblewrap` と `socat` をディストリ毎に入れる。

```bash
# Ubuntu / Debian
sudo apt-get install bubblewrap socat

# Fedora
sudo dnf install bubblewrap socat
```

### サンドボックスの有効化（最小）

`.claude/settings.json` に書く。

```json
{
  "sandbox": {
    "enabled": true,
    "autoAllowBashIfSandboxed": true
  }
}
```

セッション中はスラッシュコマンドで起動する：

```
/sandbox
```

### 抜け穴を塞ぐ

`allowUnsandboxedCommands` を false に、`failIfUnavailable` を true にすると、サンドボックスが効かない環境では起動を諦めさせられる。

```json
{
  "sandbox": {
    "enabled": true,
    "allowUnsandboxedCommands": false,
    "failIfUnavailable": true
  }
}
```

### ファイルシステム制限

`denyRead: ["~/"]` で家全部を遮断したうえで、`allowRead: ["."]` でカレントだけ開ける構図が原典の推奨。

```json
{
  "sandbox": {
    "enabled": true,
    "filesystem": {
      "allowWrite": ["~/.kube", "/tmp/build"],
      "denyRead": ["~/"],
      "allowRead": ["."]
    }
  }
}
```

### ネットワークの allow/deny

ドメイン単位で許可リストを作る。`WebFetch(domain:...)` を permissions 側にも書くと、Read/Bash 系の許可と整合が取れる。

```json
{
  "sandbox": {
    "network": {
      "allowedDomains": [
        "github.com",
        "api.github.com",
        "registry.npmjs.org",
        "pypi.org"
      ],
      "deniedDomains": [
        "*.example-malicious.com"
      ]
    }
  },
  "permissions": {
    "allow": [
      "WebFetch(domain:github.com)",
      "WebFetch(domain:docs.anthropic.com)"
    ]
  }
}
```
