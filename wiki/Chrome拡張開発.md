# Chrome拡張開発

## 概要

著者が 2026 年 4 月に着手した、自作 Chrome 拡張機能の開発記録。社内 LLM 関連の利便性向上と、自身の生産性向上を兼ねている。

## 本文

### 1. Gem 呼び出し Chrome 拡張 — 04/13 着手

- **きっかけ**: まじんさんの **Canvus をサイドバーから呼び出せる拡張機能** を真似て、Gem を呼び出せる Chrome 拡張を作ることに決めた。
- **狙い**: この機能により **Gem 掲示板の活用が促進されると嬉しい**、という期待。
- **位置付け**: 社内ユーザーが Gem に素早くアクセスできるサイドバー UI を提供。

> **不明:** Gem の具体的な構成（Gemini ベース／社内独自 Gem 掲示板の仕様）は raw に詳細記載なし。

### 2. TODO 登録 Chrome 拡張 — 04/17 着手

- **動機**: 自分が使って便利かどうかを判断する。
- **設計判断（04/22）**: 要望としては「TODO ではなく **予定** にしたい」「**日付・時間指定** したい」が出たが、対応すると右クリック後に **選択画面や入力画面** が出る必要が生じる。
  - それなら直接入れた方が早い → **quick に TODO 登録できなくなる** → 対応しないと判断。
  - **いったん現状のままリリース**する方針。

#### 設計上の根本ルール

> 右クリックから 1 動作で完了する quick 登録性が、本拡張の核。要望対応で UI ステップが増えるなら、それは別ツールでやるべき。
